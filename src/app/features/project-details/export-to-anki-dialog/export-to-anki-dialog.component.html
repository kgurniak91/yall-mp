How to Add Lookup Notes

  • Click a word (or drag to select a phrase) in the subtitles to open a lookup service (website).
  • Ensure the lookup service (website) is set to use the built-in browser in global settings.
  • Inside the browser, select any text and add it as a note using one of these methods:
    • Right-Click → Add to Notes
    • Pressing Ctrl+Shift+S
Note: The new note will be saved under the original word/phrase that was selected from the subtitles.
